DSP 课程综合设计

基于加速度信号处理的人体跌倒检测与运动状态识别系统

利用手机或可穿戴设备的三轴加速度数据，完成采样、滤波、合加速度计算、FFT 频谱分析、峰值检测、特征提取和规则判决，实现静止、行走、跑步、上下楼梯与跌倒状态识别。

选题背景 本课题面向智能养老、运动健康和安全预警等真实场景。人体在行走、跑步、上下楼梯、坐下站起和跌倒时，加速度信号在幅值、能量、主频和冲击峰值上存在明显差异，适合用数字信号处理方法进行分析和识别。

一

选题背景与选题原因

随着智能手机、智能手环和可穿戴设备的普及，人体运动数据采集变得非常方便。跌倒检测具有较高现实价值，尤其适用于老人、独居人群、运动人群和特殊作业场景。该系统以三轴加速度信号为研究对象，通过 DSP 方法提取运动特征并识别异常跌倒事件。

生活化程度高

手机传感器即可采集数据，可记录静止、行走、跑步、上下楼梯、坐下站起、模拟跌倒等动作。

课程覆盖全面

覆盖采样定理、数字滤波器、FFT、窗函数、峰值检测、特征提取和分类判决等知识点。

结果展示直观

三轴波形、合加速度、频谱图、峰值标注、运动时间线和跌倒告警都可以直接可视化。

二

项目目标

  • 读取或模拟手机三轴加速度数据，支持 CSV 数据导入。
  • 对原始信号进行去噪、归一化、平滑和低通滤波处理。
  • 计算三轴合加速度 A = sqrt(ax² + ay² + az²)，构建运动强度信号。
  • 对信号进行分帧加窗，提取均值、方差、RMS、峰值、峰峰值、主频和频谱能量。
  • 使用 FFT 识别行走、跑步等周期性运动的主频特征。
  • 设计基于冲击阈值、跌倒后静止时间和能量变化的跌倒检测规则。
  • 实现静止、行走、跑步、上楼梯、下楼梯、坐下站起和跌倒状态识别。
  • 开发可视化监测平台，显示波形、频谱、特征参数、状态时间线和跌倒告警。

系统设计方案

系统从数据采集开始，经过信号预处理、滤波、频域分析和特征提取，最终输出运动状态和跌倒告警。

主要功能

  • 内置多种运动场景模拟数据，也支持 CSV 上传。
  • 显示三轴加速度、合加速度、滤波曲线和跌倒峰值。

五

关键算法设计

1. 合加速度

将三轴加速度合成为一个方向无关的运动强度信号。

A[n] = sqrt(ax[n]^2 + ay[n]^2 + az[n]^2)

2. 低通滤波

人体日常运动主要集中在低频段，使用移动平均/低通滤波抑制高频噪声。

y[n] = (x[n] + x[n-1] + ... + x[n-M+1]) / M

3. FFT 主频

对加窗后的分析窗口进行 FFT，提取能量最大的有效频率作为主频。

f_main = argmax |FFT(A_window)|, 0.2Hz ≤ f ≤ 8Hz

4. 跌倒判决

同时满足冲击峰值、跌倒后静止和短时能量变化条件时判定为跌倒。

peak(A) > T_impact 且 after_RMS < T_still 且 still_time ≥ 0.8s

八

总结与创新点

本系统将 DSP 技术应用于人体安全监测场景，使用普通手机传感器即可完成数据采集和实验验证。项目完整体现了从采样、预处理、滤波、频谱分析到特征提取和状态判决的数字信号处理流程，同时通过可视化平台直观展示算法效果，适合作为数字信号处理课程期末大作业和答辩展示。

实用性

面向跌倒检测、智能养老、运动监测和可穿戴安全提醒等真实需求。

完整性

同时实现数据生成、CSV 导入、滤波、FFT、特征提取、状态识别和告警日志。

扩展性

后续可加入陀螺仪姿态角、机器学习分类器、云端历史统计和移动端部署。
